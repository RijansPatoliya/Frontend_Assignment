# WordPress Integration Guide (Step-by-Step)

Aapko WordPress bilkul nahi aata, fir bhi aap is landing page ko WordPress par **sirf 5 minute me** live kar sakte hain. Neeche diye gaye steps ko follow karein:

---

## Step 1: Apne Computer Par WordPress Install Karein (Free & 1-Click)

WordPress ko locally chane ke liye hum **LocalWP** naam ke software ka use karenge, jo sabse aasan hai.

1. **LocalWP Download Karein**:
   * [https://localwp.com/](https://localwp.com/) par jaakar **Download for Free** par click karein.
   * Apni details bharein (koi card/payment nahi chahiye, bilkul free hai) aur installer download karke install kar lein.

2. **Ek New Website Banayein**:
   * LocalWP software open karein aur click karein: **"+" (Create a new site)** button par.
   * Apni site ka naam rakhein (jaise: `bolly-shampoo`) aur **Continue** karein.
   * **Preferred** settings par click karein aur **Continue** dabayein.
   * Ek **WordPress Username** aur **Password** set karein (isse yaad rakhiyega, jaise username: `admin`, password: `adminpassword`).
   * **Add Site** par click karein. 1 minute me WordPress automatic install ho jayega!

3. **Admin Dashboard Open Karein**:
   * Site create hone ke baad LocalWP me right side par **"WP Admin"** button hoga. Uspe click karein.
   * Ek tab khulegi, wahan apna set kiya hua Username aur Password daal kar login karein.

---

## Step 2: Elementor Page Builder Install Karein

Elementor WordPress ka sabse popular visual editor hai jo hamare custom code ko directly handle kar sakta hai.

1. WordPress Dashboard me left menu me **Plugins -> Add New** par click karein.
2. Search bar me type karein: `Elementor`.
3. **Elementor Website Builder** ke saamne **Install Now** par click karein, fir install hote hi **Activate** par click kar dein.

---

## Step 3: Landing Page Code Import Karein (Copy-Paste)

Kyunki maine aapke liye ek single complete self-contained file (`wordpress-elementor-code.html`) bana di hai, aapko koi file uploads ya paths change nahi karne hain.

1. WordPress Dashboard me left menu me **Pages -> Add New** par click karein.
2. Page ka Title rakhein: `bolly - Knock Out Flakes`.
3. Edit window me upar bade blue button **"Edit with Elementor"** par click karein.
4. Elementor side panel me search box me type karein: `HTML`.
5. **HTML widget** ko drag karke page builder ke body block me drop karein.
6. Is repository me bani **[wordpress-elementor-code.html](file:///c:/Users/Khushi/Desktop/comapny_assignment/wordpress-elementor-code.html)** file ko open karein, iske saare code ko copy (`Ctrl + A` fir `Ctrl + C`) karein.
7. Elementor ke side panel me **HTML Code** block me pura copy kiya hua code paste (`Ctrl + V`) kar dein.
8. **Boom!** Aapka interactive 3D bottle wala page screen par render hona shuru ho jayega.
9. Click **Publish** at the bottom left. Aapka page WordPress par live hai!

## Step 4: Screen Recording Kaise Karein (Submission Requirement)

Assignment details ke hisab se aapko 3-5 minute ka video banana hai:
1. Apne browser me is live page ko open karein.
2. Ek free tool jaise **Loom** (loom.com) ya **OBS Studio** download karein.
3. Desktop screen record karte hue:
   * Page ko scroll karke beautiful premium responsive layout dikhayein.
   * Apne mouse se 3D bottle ko drag karke rotate karke dikhayein.
   * Bottom me color dots ko click karke 3D bottle ka color change hote hue dikhayein.
   * Mobile mode inspect karke mobile view bhi record karein (drag rotate & sticky nav scroll).
4. Video ko Google Drive ya Loom par upload karein aur share link (with view permissions) ko compile karein.
